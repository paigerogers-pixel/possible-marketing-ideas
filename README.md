# Marketing Ideas - Editorial Website

A modern, editorial-style website built with HTML, CSS, and JavaScript, hosted on GitHub Pages.

## Features

- **Editorial Design**: Clean, professional layout with serif typography for a magazine-like feel
- **Responsive Layout**: Fully responsive design that works on desktop, tablet, and mobile devices
- **Smooth Navigation**: Sticky navigation bar with smooth scrolling between sections
- **Content Sections**: 
  - Hero section with compelling headline
  - About/Project introduction
  - Core strategy pillars with 3-column grid
  - Key insights with 4-card layout
  - Contact form
- **Professional Styling**: Modern color scheme with accent colors and subtle animations
- **Accessible**: Semantic HTML and keyboard navigation support

## Getting Started

1. Clone this repository
2. Enable GitHub Pages in your repository settings
3. Navigate to `https://your-username.github.io/possible-marketing-ideas/` to view your site

## Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #1a1a1a;
    --accent-color: #c9302c;
    --light-bg: #f8f8f8;
    /* ... etc */
}
```

### Content
Edit the HTML sections in `index.html` to add your own:
- Hero section text
- About/strategy content
- Insight cards
- Contact information

### Typography
Modify the font families in `styles.css`:
- `--font-serif`: Editorial headings
- `--font-sans`: Body text and UI
